# 🧠 MediScan AI – Intelligent Disease Prediction System

MediScan AI is a machine learning-based application that predicts diseases based on user input data. It helps in early diagnosis and provides useful insights.

---

## 🚀 Features

* Disease prediction using Machine Learning
* Fast backend using Flask
* Stores data using SQLite

---

## 🛠️ Tech Stack

* Python
* Flask
* Machine Learning
* SQLite

---

## ⚙️ How to Run

1. Clone the repository
   git clone https://github.com/Ushapriya06/mediscan-ai.git

2. Go to project folder
   cd mediscan-ai

3. Install dependencies
   pip install -r backend/requirements.txt

4. Run the app
   python backend/app.py
